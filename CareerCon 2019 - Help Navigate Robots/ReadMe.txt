Robots are smart… by design. To fully understand and properly navigate a task, however, they need input about their environment.

In this competition, you’ll help robots recognize the floor surface they’re standing on using data collected from 
Inertial Measurement Units (IMU sensors).

https://www.kaggle.com/c/career-con-2019

Approaches:
Feature engineering for attrubute selection
A Simple Neural Network model (overfitted) and Random Forest (better results, 87% test accuracy)
